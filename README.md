#学生选课系统 计181 2018310735 黄宁烨


一、实验目的

1、分析学生选课系统

2、使用GUI窗体及其组件设计窗体界面

3、完成学生选课过程业务逻辑编程

4、基于文件保存并读取数据

5、处理异常

二、实验要求

1、	设计GUI窗体，支持学生注册、课程新加、学生选课、学生退课、打印学生选课列表等操作。

2、	基于事件模型对业务逻辑编程，实现在界面上支持上述操作。

3、	针对操作过程中可能会出现的各种异常，做异常处理。

4、	基于输入/输出编程，支持学生、课程、教师等数据的读写操作。
